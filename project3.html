Project Description

This project's objective was to "design and code an experimental clock that expresses a novel way of tracking time." I wanted to create a sort of "clock" that keeps track of the moon's phases. This Sketch shows the current moon phase, what it looks like, a countdown till the next moon phase and what that next phase will be.
